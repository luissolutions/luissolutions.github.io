Triforce Servers — Mod Setup

Step-by-Step:


  1. Download and install CurseForge App.
  2. In CurseForge, browse Minecraft modpacks and install the one matching the Triforce server.
  3. Download and run MinecraftClientUpdater.exe — it automatically syncs your mods folder with the server's current mod list.
  4. Launch Minecraft through CurseForge and log in with your Microsoft account.
  5. Connect to oddohome.asuscomm.com:25566 (Triforce) or oddohome.asuscomm.com:25565 (Triforce 2).

Run MinecraftClientUpdater.exe any time the server updates mods — it pulls the latest from oddohome.asuscomm.com:90/mods/.
