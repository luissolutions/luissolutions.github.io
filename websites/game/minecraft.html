Triforce Servers — Mod Setup

  1. Install CurseForge App.
  2. Browse Minecraft modpacks and install the one matching the Triforce server.
  3. Run MinecraftClientUpdater.exe to sync your mods folder.
  4. Log in with your Microsoft account.
  5. Connect to oddohome.asuscomm.com:25566 or :25565 for Triforce 2.

Re-run MinecraftClientUpdater.exe whenever the server updates mods.
