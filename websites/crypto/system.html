Watchlist Setup — 5 Steps

  1. Create Categories — Add the eight core Eco Lou categories.
  2. Populate Companies — Use public data sources to add names to each category.
  3. Track Key Metrics — Revenue growth, free cash flow, dividend growth, payout ratio, debt, ROIC.
  4. Review Quarterly — Earnings, cash flow, dividend announcements, guidance updates.
  5. Maintain Balance — Avoid over-concentration in any single category.
