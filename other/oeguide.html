Creating a USB Recovery Drive

Reference: OpenEye Guide — Creating a USB Recovery Drive

  1. Check image version in OWS (Help → About) and download the correct ISO.
  2. Download and run Rufus.
  3. Select ISO and configure:
    • Linux — FAT32, GPT, UEFI
    • Windows — NTFS, GPT, UEFI
  4. Start, confirm wipe, and wait for completion.
  5. Eject USB safely.
System Recovery/Reinstall via USB

Reference: OpenEye Guide — System Recovery via USB

  1. Insert USB into USB port.
  2. Reboot & press correct boot key.
    • Most Models - F12 key
    • MV model units - F7 key
    • MD model units - ESC to get to the BIOS. Hit ESC after exiting to load the boot menu
  3. Password if prompted: dvr4321.
  4. Select UEFI USB from boot menu.
  5. Follow on-screen instructions to complete recovery.
  6. Enter Serial Number for Server
  7. Wait for system to reboot.
Network Settings Setup
  1. Access the network settings menu.
  2. Re-enter the network settings from before the reset.
Apex Software Update Process
  1. Go to settings and select Software update
  2. Enter link to update Software https://files.openeye.net/apex/release/commercial/APX_x64_windows_3.2.6.253.exe
  3. Import config backup for that store if availible.
